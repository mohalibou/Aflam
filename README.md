# Aflam

A movie database app where you can search for movies, as well as view trending and upcoming movies.

### Frameworks:
- SwiftUI
- TMDB API

### Features:
- View trending movies.
- View upcoming movies.
- Search for movies of your choosing.
- View ratings for each movie.
